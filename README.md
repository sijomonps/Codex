# 🌐 AWS Hosted Minimal Website

A simple website built using HTML and CSS, hosted on AWS 

---

## 📌 About

This project was created for a fun and competitive class activity where:

* Teams build and host websites on AWS
* Other teams attempt to break or test the websites

We designed our site to be **minimal, simple, and stable**.

---

## 🎯 Purpose

To learn:

* Website hosting on AWS
* Basic security and stability concepts
* Real-world deployment challenges
